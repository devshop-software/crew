## Obsidian vault reference

The Obsidian vault (accessed via the `obsidian` MCP) is used to research and draft
changes to the crew skills/agents. The working copy lives at:

`projects/reusable-agentic-workflow/crew-v4/` — with `skills/` (adjust, run, pulls) and
`agents/` (implementation, qa, reviewer, mr-review, findings, pull-triage, merge-judge). The
frozen V3 design (the full pre-consolidation set, including the retired ticket / improve /
merge / approve / plan / groom skills) is at `.../archived/crew-v3/`.

The vault is sometimes used as the source for importing those changes into this repo,
so it may be ahead of or drifted from the plugin code here. Check the vault when
researching or syncing skill/agent updates.
